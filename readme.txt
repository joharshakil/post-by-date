== Description ==
A powerful post filter plugin used to display a list of posts from certain categories with date and limit filter.

== Installation ==
1- Download Posts by date pugin
2- Go to: WordPress Admin > Plugins > Add New and Upload Plugin with the file you downloaded with Choose File.
3- Install Now and Activate the extension.
